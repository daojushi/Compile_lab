**程序编译流程**
一个yacc程序变为可执行语法分析程序的步骤如下：

1. 使用yacc编译**.y**文件生成语法分析器的**.c**文件：

   ```shell
   yacc test01.y #默认生成文件为y.tab.c
   ```

2. 使用gcc等编译器编译生成可执行文件

   ```shell
   gcc y.tab.c -o 1
   ```

3. 执行即可
   ```shell
   ./1
   ```
